# Index Clustering?

---

- 테이블의 PK에 적용되는 인덱스 값임.
- 그 인덱스 값이 비슷비슷한 녀석들을 묶어서 저장하는 걸 인덱스 클러스터링이라고 함
- 따라서 PK값이 바뀌면 디스크에 저장되어있는 물리적인 위치가 바뀌게 됨
- PK를 기반으로한 검색이라면 굉장히 빠르지만, 레코드의 저장이나 PK값을 변경할 때에는 느리다.
