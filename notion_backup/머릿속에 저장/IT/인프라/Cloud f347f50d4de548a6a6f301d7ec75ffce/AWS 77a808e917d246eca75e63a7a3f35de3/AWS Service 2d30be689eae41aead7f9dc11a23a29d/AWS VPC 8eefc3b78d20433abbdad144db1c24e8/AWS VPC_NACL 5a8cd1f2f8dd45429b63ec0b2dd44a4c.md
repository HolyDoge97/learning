# AWS VPC_NACL

---

- NACL, 온프레미스에서는 ACL이라고 부르는 개념과 맞는 듯
    - 간단히, 네트워크 영역에서 인바운드/아웃바운드를 제어하는 기능(라우터 방화벽의 기능)

<aside>
💡 NACL은 Stateless한 보안 리스트이고, 보안그룹과는 차별점이 존재함
Access Block은 NACL에서만 가능하다.

</aside>
